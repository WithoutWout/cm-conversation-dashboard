How to Get Export Files

Follow these steps in the CM.com AI Cloud dashboard to download the JSON export files needed by this app.

Articles

1 In CM.com Conversational AI Cloud, open the Articles section from the left navigation.
2 Click the three dots menu and click Export.
3 Choose Export JSON and save the downloaded file.
4 Save the downloaded file in the folder you selected in this app under Settings → Data Folder.

Dialogs

1 In CM.com Conversational AI Cloud, open the Dialogs section from the left navigation.
2 Click the three dots menu and click Export.
3 Choose Export JSON and save the downloaded file.
4 Save the downloaded file in the folder you selected in this app under Settings → Data Folder.
